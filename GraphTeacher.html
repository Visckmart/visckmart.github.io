GraphTeacher

A graph creation and algorithm exploration tool

Math Education

This is an educational tool that helps students to learn about Discrete Math's graphs. In this context, graphs are mathematical structures that can be used to model real-life problems, and solve them by using the appropriate algorithms.

GraphTeacher is a tool that enables students to create their own graph with a very intuitive interface, run a set of algorithms, and learn about how they work with step by step interactive instructions.

Technologies

The tool is available on the web via a static page. It was developed using HTML, CSS and JavaScript without any external libraries or frameworks.

We decided to implement everything from the ground up as a challenge. This meant that all of the gesture handling, the graph's visualization, the graph algorithms, the internal structures and the serialization that enable sharing were entirely implemented by us.

Methodology

This project began being developed during the college vacation period between 2020 and 2021. It took only around 3 months for it to be almost complete, with all the features implemented and only small adjustments being necessary.

On the starting point, we agreed on what the tool was going to be. Right after that we had a brainstorm and made a list of every idea we thought was cool to have implemented, along with its priority level and how hard it would be to implement it.

After that we selected the most essential features, prepared a git repository and assigned the implementation tasks between ourselves. On the following weeks we kept talking with eachother almost daily, asking about implementaiton details and best practices, and talking about our progress.

This dynamics worked out really well and we kept assigning the tasks that came from the brainstorm until the college classes started and we slowed down the development, as planned.
